# 🚀 Deploy para Vercel (2 Cliques)

## Passo 1
Clique neste link:
```
https://vercel.com/new/clone?repository-url=https://github.com/GabrielNascimentoL/teste-claude&project-name=clawdbot-landing
```

## Passo 2
Na página da Vercel, clique em **"Deploy"**

## Pronto! ✅
Sua landing page estará online em segundos.

## Detalhes do Projeto

- **Landing Page:** ClawdBot - seu assistente IA pessoal
- **Design:** Moderno, responsivo, sem dependências
- **Stack:** HTML + CSS puro (zero JavaScript)

## Customizar

Edite `index.html` diretamente no GitHub ou clone localmente:

```bash
git clone https://github.com/GabrielNascimentoL/teste-claude.git
cd teste-claude
# Edite index.html
git add .
git commit -m "Customize landing page"
git push
```

A Vercel fará redeploy automaticamente!
