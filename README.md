# ClawdBot Landing Page 🦾

Uma landing page bonita e responsiva que apresenta os benefícios do ClawdBot.

## Deploy Rápido

### Vercel
Clique no botão abaixo para fazer deploy em 1 clique:

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/GabrielNascimentoL/teste-claude)

### Ou manualmente:
1. Vá para https://vercel.com
2. Importe este repositório
3. Clique em Deploy
4. Pronto! 🎉

## Local

```bash
# Apenas abra index.html em um navegador
open index.html
```

## Features

- Design moderno e responsivo
- Tailwind CSS custom
- Sem dependências
